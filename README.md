# 💼 Sistema de Gerenciamento - Doufer

📋 **Descrição**

Este é um sistema desktop desenvolvido exclusivamente para a loja **Doufer**, com foco no gerenciamento de **clientes** e seus **pedidos**. A aplicação permite realizar o cadastro, edição, consulta e exclusão de clientes, bem como o controle detalhado dos pedidos vinculados a cada cliente. Também é possível adicionar fotos aos pedidos e excluir pedidos individualmente.

O sistema foi criado para uso interno da loja, utilizando tecnologias modernas baseadas em JavaScript e rodando tanto como aplicação desktop quanto via navegador, para facilitar o desenvolvimento.

---

### ▶️ Como executar

#### 💻 Modo Desktop (Electron)

1. Instale as dependências:
   ```bash
   npm install
````

2. Execute o sistema:

   ```bash
   npx electron main.js
   ```

---

#### 🌐 Modo Navegador (Desenvolvimento)

1. Acesse a pasta do backend:

   ```bash
   cd src/backend
   ```

2. Inicie o servidor:

   ```bash
   node server.js
   ```

3. Abra o navegador em:

   ```
   http://localhost:3000
   ```

---

### 🛠️ Tecnologias usadas

* Node.js
* JavaScript
* Electron
* Express.js
* Sequelize ORM
* SQLite
* Handlebars (template engine)

---

### ✅ Funcionalidades (em desenvolvimento)

* Cadastro de clientes
* Cadastro de pedidos vinculados aos clientes
* Consulta de clientes
* Visualização dos pedidos no perfil do cliente
* Edição de dados dos clientes
* Exclusão de clientes
* Exclusão de pedidos
* Adição de fotos aos pedidos
